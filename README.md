# cwlindean-homepage

个人主页，两种设计风格可选。

## 预览

| 风格 | 文件 | 说明 |
|---|---|---|
| 留白杂志风 | `cwlindean-editorial.html` | MUJI / Kenya Hara 风格，暖纸底色，衬线标题，编号列表，极简装饰 |
| 温暖人文风 | `cwlindean-warm.html` | Stripe Press / Mailchimp 风格，暖骨色基调，独立强调色卡片，有机圆角 |

## 项目

- 云盘 — cloud.cwlindean.top
- AI 对话 — chat.cwlindean.top
- NewAPI — api3.cwlindean.top
- 观影 — movie.cwlindean.top
- 剪贴板 — netcut.cwlindean.top
- 文件分享 — file.cwlindean.top

## 部署

纯静态 HTML，可直接部署到任意托管服务：

```bash
# Cloudflare Pages / Vercel / Netlify
# 选择 cwlindean-editorial.html 或 cwlindean-warm.html 作为入口
# 或重命名为 index.html
```
